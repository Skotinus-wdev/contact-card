# Digital Contact Card

A simple, elegant digital contact card for AI event networking.

## Features
- QR code leading to contact page
- Professional photo display
- Direct contact links (WhatsApp, Telegram, Email)
- One-click messaging (opens respective apps)
- Mobile-optimized design

## Tech Stack
- Next.js 14 + TypeScript
- Tailwind CSS
- Static export for easy hosting

## Contact Links Format
- WhatsApp: `https://wa.me/[number]`
- Telegram: `https://t.me/[username]`
- Email: `mailto:[email]`

## QR Code
Generated QR code points to deployed website URL.
